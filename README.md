# KSU Wallet

## Description
This project is a basic digital wallet system developed using Python, Tkinter, and a SQLAlchemy-backed SQLite database.
It allows students at King Saud University to register, log in, and view their wallet balance through a graphical user interface.
An administrator panel is included, enabling administrators to manage student balances and add new university entities from a separate control window.
All application data is stored in a local database file named `Database.db`.

## Features
- Student registration and login system
- Digital wallet balance viewing
- Administrator panel for managing balances
- Ability to add and manage university entities
- Database-backed data persistence

## Technologies Used
- Programming Language: Python
-  GUI Framework: Tkinter
- Database: SQLite
- ORM: SQLAlchemy
